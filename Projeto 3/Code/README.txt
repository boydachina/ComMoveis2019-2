Estes programas foram utilizados Ubuntu 18.04 LTS.
Código feito em python3.
Requisitos para o funcionamento dos programas a seguir:
-ter instalado o pip:
	Para derivados do Debian:
	sudo apt install python3-pip
	Para fedora:
	sudo dnf install python3
	Para CentOS:
	sudo yum install epel-release
	sudo yum install python-pip
-Depois de instalar o pip, é necessário instalar a biblioteca numpy,matplotlib e a commpy
	para instalar a biblioteca numpy:
	pip3 install numpy
	para instalar a biblioteca matplotlib:
	pip3 install matplotlib
	para instalar a biblioteca commpy:
	pip3 install scikit-commpy
Para executar os programas a seguir pode entrar em uma ide de sua preferência ou executar diretamente do terminal (na pasta que baixou os arquivos) com os comandos:
	python3 BEROFDM.py
pode demorar algum tempo devido a quantidade de bits
