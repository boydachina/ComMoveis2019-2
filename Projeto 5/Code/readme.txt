Estes programas foram utilizados Deepin 15.
Código feito em python3.
Requisitos para o funcionamento dos programas a seguir:
-ter instalado o pip:
	Para derivados do Debian:
	sudo apt install python3-pip
	Para fedora:
	sudo dnf install python3
	Para CentOS:
	sudo yum install epel-release
	sudo yum install python-pip
-Depois de instalar o pip, é necessário instalar a biblioteca numpy,matplotlib,pandas e a Si prefix
	para instalar a biblioteca numpy:
	pip3 install numpy
	para instalar a biblioteca matplotlib:
	pip3 install matplotlib

python3 posgrid.py
